# Design Reference Notes - AI Creative Agency

## Key Design Elements Observed:

### Color Scheme
- Pure black background (#000000)
- White text for headlines
- Orange/coral accent color for CTAs and highlights
- Subtle gray for secondary text

### Typography
- Large serif font for main headlines (similar to Instrument Serif)
- Clean sans-serif for body text
- Very large headline sizes (hero text is massive)

### Layout
- Full-width hero with centered content
- Image grid/gallery below hero showing AI-generated work
- Asymmetric layouts for problem/solution sections
- Clean, minimal navigation

### Visual Elements
- Rounded pill-shaped badge at top ("Custom AI Images & Videos For Your Business")
- Orange/coral CTA button ("Message Me on WhatsApp")
- Grid of AI-generated images showcasing capabilities
- Large imagery paired with text sections
- Accordion-style FAQ section

### Animations & Effects
- Clean, minimal animations
- Smooth scrolling
- Hover effects on images

### Key Sections
1. Hero with badge, headline, subheadline, CTA, image grid
2. Problem section with large image and text
3. Services/capabilities section
4. Process section
5. FAQ accordion
6. Final CTA

### Style Characteristics
- Very dark, premium feel
- High contrast (white on black)
- Orange accent creates warmth
- Showcases actual AI work prominently
- Less glassmorphism, more solid colors
- Bolder, more direct design
