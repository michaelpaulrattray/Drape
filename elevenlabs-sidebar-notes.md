# ElevenLabs Sidebar Reference Notes

## Expanded State (~140px wide sidebar)
- Top: Brand logo "ElevenLabs" + workspace name "ElevenCreative" with switcher icon
- Section 1 (ungrouped): Home, Voices (+), Studio (highlighted/active), Flows (New badge), Files
- Section 2 "Pinned": Text to Speech, Sound Effects, Image & Video, Voice Isolator, Voice Changer, Music, Speech to Text, More tools (expandable)
- Bottom: "Invite team members" card, "Developers" link
- Active item has subtle highlight background
- Clean, minimal — no borders, just spacing and font weight
- White/light background, black text, orange accent dot for brand

## Collapsed State (~48px wide sidebar)
- Top: Toggle button (pause/hamburger icon), brand icon (orange dot)
- Icons only, vertically stacked, matching the expanded order
- Same icon set: Home, Voices, Studio, Flows, Files, then pinned tools
- Bottom: chat/support icon
- No labels, no section headers
- Same white background

## Top Bar (both states)
- Left: small icon + "Studio" breadcrumb text
- Right: Feedback, Docs, Ask, notifications, user avatar
- Minimal height, clean

## Main Content Area
- "Studio" heading with Upload + New blank project buttons
- "Get Started" horizontal card carousel
- "Recent Projects" with search, filters, grid/list toggle
- Cards show project thumbnails with titles

## Key Behaviors
- Toggle button at top-left of collapsed rail expands/collapses
- Sidebar pushes content (not overlay)
- Active tool highlighted in sidebar
- Collapsed state preserves icon order from expanded
