# Dashboard Reference Notes - Mango (Streaming Channel Dashboard)

## Key Design Elements:

### Layout Structure
- Left sidebar with navigation (collapsible)
- Main content area with header banner
- Search bar in top header
- User avatar in top right

### Color Scheme
- Pure black/very dark background (#0a0a0a or similar)
- Lime/yellow-green accent color for active states and highlights
- White text on dark backgrounds
- Subtle gray borders and dividers

### Sidebar Navigation
- Logo at top
- Icon + text navigation items
- Sections: Main nav, "My Channel", "Subscriptions"
- Active state with lime/green highlight
- "Show More" expandable section
- Subscription avatars with channel names

### Header/Banner Area
- Large hero banner image
- Profile avatar overlapping banner
- Channel name and subscriber count
- Action buttons (Join, Subscribed)
- Tab navigation (Home, Videos, Shorts, Live, Playlists, Community, Channels, About)

### Content Cards
- Dark card backgrounds
- Thumbnail images with rounded corners
- Video duration badges
- Title, view count, time ago
- CC badge for closed captions

### Top Bar
- Search input with icon
- Notification icons
- User avatar

### Key UI Patterns
- Rounded corners on cards and buttons
- Lime/green accent for primary actions
- Subtle hover states
- Clean, minimal iconography
